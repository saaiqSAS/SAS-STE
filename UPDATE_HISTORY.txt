THESE ARE THE UPDATE HISTORY OF SAS-STE
==========================================

WHAT THE PLACE VALUES OF VERSION NUMBER MEAN:

vX.Y.Z

Z= increases after a bug fix
Y= increses after adding a new feature
X= increases after an encryption algorithm change

===========================================

SAS-STE-Wizard:
------------------

v3.3.5 :
 > Ability to change static keys in the compiled version added
 > StaicKeyGen executable added to programFiles directory
 > StaticKeys.stkey files made to store static keys
 > Static keys are stored in a file instead of being embedded
 > No need to type '\' (backward slash) twice anywhere in the program
 > Feature to save output of encrypted/decrypted files to themselves
 > User security improved
 > Major bug fixes
 > Efficiency improved

v2.2.4 :
 > Encryption algorithm improved and extra security layers added
 > Support for keys added
 > Static Key and Public Key added
 > .skey file extention now used for SAS-STE key files
 > Reusing existing key feature added
 > Ignoring unknown characters feature added
 > Multiple time encryption feature added
 > Improved handling of user side errors
 > Major bug fixes
 > Efficiency improved
